## Overview
This web app is built using Flask and deployed on Heroku.<br>
It predicts flight fare based on Airline, Duration, No of Stops, Source and Destination.


## Directory Structure
Directory structure should look like this:
```
├── static
│   ├── styles.css
├── templates
│   ├── flight_fare.html
│   ├── result.html
├── app.py
├── Procfile
├── Readme.md
├── requirements.txt
├── rf_flight.pkl
```
